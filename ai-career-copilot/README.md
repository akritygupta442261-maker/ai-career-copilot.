# AI Career Copilot 🧠

AI Career Copilot is an **agentic GenAI system** that evaluates resumes against job descriptions to assess **interview readiness**.

The system combines **hybrid retrieval (RAG)**, **deterministic rule-based evaluation**, and **LLM-powered agents** to deliver **realistic, fresher-safe, recruiter-aligned feedback**.

This is **not a chatbot** — it is a **decision-oriented AI system** designed to simulate how resumes are evaluated in real hiring pipelines.

---

## 🚀 Key Features

### 🔍 Hybrid Resume Evaluation
- Skill-based scoring aligned with job descriptions  
- Experience scoring using rule-based logic  
- ATS structure validation (format, sections, clarity)

---

### 🤖 Agentic Architecture
- **Deterministic agents**
  - Resume parsing
  - Skill & experience scoring
  - ATS rule validation
- **LLM agents**
  - Resume improvement suggestions
  - Interview question generation
  - Answer refinement

Each agent has a **clearly defined responsibility**, enabling modular and explainable evaluation.

---

### 🧠 Hybrid Retrieval (RAG)
- Resume-aware chunking strategy  
- **HYDE** (Hypothetical Resume / Profile Generation) for better semantic recall  
- **MMR** for diversity-aware evidence selection  
- **Dense + sparse retrieval**
  - Dense: semantic similarity
  - Sparse: keyword and ATS-aligned matching

---

### 🎯 Dynamic Role Detection
- Job description intent detection  
- Resume strength analysis  
- Adaptive role inference  
  - Example: *Backend Engineer – GenAI Aligned*

---

### ✍️ Resume Improvement
- Generates **resume-ready bullet points**
- Suggestions grounded strictly in provided resume data  
- **Fresher-safe design**
  - No fabricated metrics
  - No fake experience injection

---

### 🎤 Interview Preparation
- Resume + JD grounded interview questions  
- Technical and behavioral coverage  
- Concise, realistic answers with **strict length control**
- Designed for real interview expectations, not verbose explanations

---

### 🖥️ Interactive UI
- Streamlit-based interface  
- Resume upload and evaluation dashboard  
- Structured feedback presentation

---

## 🔑 Key Highlight

> ❗ AI Career Copilot is **not a ChatGPT wrapper**.

It integrates:
- Rule-based evaluation pipelines  
- Hybrid retrieval strategies  
- Agent-based task separation  
- Controlled LLM usage for decision support  

This makes the system **predictable, explainable, and production-oriented**.

---

## 🎯 Intended Use Cases

- Fresher and early-career resume evaluation  
- JD-specific interview preparation  
- Learning agentic AI system design  
- Demonstrating real-world GenAI architecture in portfolios
